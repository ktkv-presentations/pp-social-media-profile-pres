CSS переменные

Коробочка с названием, в котором хранится значение (цвет, шрифт и так далее)


--my-color: #3498db;  /* Объявили переменную */
background: var(--my-color);  /* Используем её */

Название цветов

Название цветов указывается с помощью RGB или названия цвета

HEX

#FF00FF

  • # - обозначение начала Hex кода
  • FF - красный канал
  • 00 - зеленый канал
  • FF - зеленый канал


color: #FF00FF /* Розовый цвет */
